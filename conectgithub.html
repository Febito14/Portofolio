Cara Menghubungkan Project ke GitHub

Ikuti langkah-langkah di bawah ini untuk menghubungkan proyek Anda ke GitHub secara online.

Langkah 1: Membuat Repository di GitHub

Buka GitHub, login, dan buat repository baru dengan mengklik "New".

git init – Inisialisasi Git di folder proyek lokal Anda.
git remote add origin https://github.com/username/nama-repository.git – Tambahkan repository remote GitHub.

Langkah 2: Push Proyek Anda ke GitHub

Setelah proyek Anda siap, gunakan perintah berikut untuk mengunggah kode Anda ke GitHub:

git add . – Mengambil semua file untuk di-commit.
git commit -m "Initial commit" – Commit perubahan yang sudah dilakukan.
git push -u origin master – Push kode ke repository GitHub.
git status – Melihat status file anda yang sudah di masukan ke repository GitHub.

Langkah 3: Verifikasi Repository Anda

1. Kembali ke repository Anda di GitHub dan pastikan semua file telah berhasil diunggah.

2. Masuk ke Setting -> lalu pilih bagian page -> pilih bagian branch -> pilih yang Main -> save

3. Tunggu beberapa menit 1-2 menit nanti akan terlihat link project anda
