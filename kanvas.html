Elemen HTML untuk Kanvas

Berikut ini adalah beberapa elemen HTML yang digunakan dalam pembuatan kanvas

<canvas> - Elemen untuk membuat area gambar atau grafis di halaman HTML menggunakan JavaScript.

<img> - Elemen untuk menampilkan gambar di halaman HTML. Gambar ini dapat digunakan sebagai sumber untuk digambar di kanvas.

<button> - Elemen untuk membuat tombol yang dapat diklik. Dalam kasus ini, tombol digunakan untuk menjalankan fungsi yang menggambar gambar di kanvas.

<input type="file"> - Elemen input yang memungkinkan pengguna memilih file dari perangkat mereka, seperti gambar untuk ditampilkan di kanvas.



Syntax Code Kanvas HTML

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<title>Kanvas</title>
</head>
<body>
<h1>Tutorial Kanvas</h1>
<input type="file" id="fileInput" accept="image/*" />
<div>
<img id="gambar" alt="Gambar tidak tersedia" style="display: none" />
<canvas id="kanvas" width="220" height="277"></canvas>
<button onclick="kanvas()">Try</button>
</div>
</body>
</html>
