Register

Lakukan pendaftaran akun untuk melakukan Login

HTML Element Form Register

<form> - Digunakan untuk mendefinisikan form HTML. Semua input berada di dalam elemen ini.

<label> - Elemen untuk mendefinisikan label bagi input, membantu pengguna mengerti input yang diperlukan.

<input type="text"> - Digunakan untuk menerima teks dari pengguna (misalnya, nama pengguna).

<input type="email"> - Input untuk menerima email pengguna, dilengkapi dengan validasi.

<input type="password"> - Input untuk menerima kata sandi, teks yang dimasukkan akan tersembunyi.

<input type="submit"> - Tombol untuk mengirim form yang telah diisi oleh pengguna.

<input type="reset"> - Tombol untuk mengatur ulang atau mereset semua input di dalam form.

<input type="button"> - Tombol yang dapat melakukan tindakan lain, misalnya mengarahkan ke halaman login.
